# Tort House Zakaz Android APK

Bu loyiha PWA fayllarni Android WebView ichiga joylaydi.

## APK olish
1. Android Studio o'rnating.
2. Ushbu papkani Android Studio orqali oching.
3. Build > Build Bundle(s) / APK(s) > Build APK(s).
4. Tayyor APK: `app/build/outputs/apk/debug/app-debug.apk`.
