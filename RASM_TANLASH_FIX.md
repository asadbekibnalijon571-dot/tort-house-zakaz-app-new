# Rasm tanlash tuzatildi

Bu versiyada Android WebView ichida `<input type="file">` ishlashi uchun `onShowFileChooser` qo'shildi.

Yangilash tartibi:
1. GitHub repository'da `app/src/main/java/uz/torthouse/zakaz/MainActivity.java` faylini shu versiyadagi fayl bilan almashtiring.
2. Yoki butun ZIP ichidagi fayllarni repository'ga qayta upload qiling.
3. Actions avtomatik APK chiqaradi.
4. Eski APK'ni telefondan o'chirib, yangi APK'ni o'rnating.
